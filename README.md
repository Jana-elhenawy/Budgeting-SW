# 💰 Budgeting-SW

A desktop budgeting application built with **Java** and **JavaFX**, designed to help users manage their personal finances.

---

## 🖥️ Tech Stack

| Technology | Purpose |
|---|---|
| Java 25 | Core programming language |
| JavaFX 21 | Desktop UI framework |
| JUnit 5 | Unit testing |
| Maven | Build & dependency management |
| ControlsFX | Enhanced UI controls |
| BootstrapFX | UI styling |
| Ikonli | Icon library |
| TilesFX | Dashboard tile components |

---

## 📋 Prerequisites

Before running the project, make sure you have the following installed:

- **Java JDK 21+** — [Download here](https://adoptium.net/)
- **Maven 3.8+** — [Download here](https://maven.apache.org/download.cgi)
- **IntelliJ IDEA** (recommended) or any Java IDE

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Jana-elhenawy/Budgeting-SW.git
cd Budgeting-SW
```

### 2. Install dependencies
```bash
mvn install
```

### 3. Run the application
```bash
mvn clean javafx:run
```

---

## 🧪 Running Tests

The project uses **JUnit 5** for unit testing.

```bash
mvn test
```

You should see output like:
```
Tests run: X, Failures: 0, Errors: 0, Skipped: 0
BUILD SUCCESS
```
